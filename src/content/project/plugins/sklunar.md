**SkLunar** is a [Skript](https://github.com/SkriptLang/Skript) addon that adds support for the Lunar Apollo API.

## Dependencies
- [Skript](https://github.com/SkriptLang/Skript)
- [Apollo](https://lunarclient.dev/apollo/downloads)
